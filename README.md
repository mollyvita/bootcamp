# 🎓 VuzTime — Агрегатор университетских мероприятий

VuzTime помогает студентам и абитуриентам находить интересные события в ведущих вузах (МГУ, МФТИ, ВШЭ, МГТУ, ИТМО).

**🔗 Ссылка на проект:** [https://bootcamp-psi-steel.vercel.app/](https://bootcamp-psi-steel.vercel.app/)

## ✨ Основные функции
- **Каталог мероприятий**: Фильтрация по вузам, типу события и целевой аудитории.
- **Умная регистрация**: Простой процесс записи на лекции, мастер-классы и дни открытых дверей.
- **ИИ-ассистент (Gemini)**: Персональные рекомендации на основе запросов пользователя.
- **Личный кабинет**: Просмотр списка своих регистраций.

## 🛠 Технологический стек
- **Backend**: Python, Flask
- **AI**: Google Gemini API (gemini-2.5-flash-lite)
- **Frontend**: HTML5, Vanilla CSS (Glassmorphism), JavaScript
- **Deployment**: Vercel

## 🚀 Как запустить локально
1. Клонируйте репозиторий.
2. Установите зависимости: `pip install -r requirements.txt`.
3. Создайте файл `.env` и добавьте `GOOGLE_API_KEY`.
4. Запустите приложение: `python app.py`.
