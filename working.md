# План изменений

- Используй логотип из logo.jpeg для сайта
- Сделай рабочий профиль пользователя
- В профиле нужны "Мои записи" (которые сейчас отдельно) и настройки
- В настройках можно будет поменять сохранённые данные (ФИО, почта, номер)
- Эти данные будут автоматически заполняться при регистрации на мероприятия
- В чате с ИИ-ассистентом, если тот хочет предложить мероприятия, вместо простого Markdown (который, к слову, не работает) надо выводить красивые плашки с предложением зарегистрироваться
- Меню сверху можно убрать, ИИ-ассистента показывать сразу за цифрами (кол-во меро, университетов и категорий)

- Если не отправляешь - ассистент является небольшим окном, пролистав мимо которого будет список, как и было
- Проще говоря, сайт должен приоритизировать ИИ и быть похожим на известные сайты с ИИ (Lovable, например)
- В список мероприятий нужно добавить мероприятия с известных иностранных вузов, а также фильтр по стране
- Добавь меню выбора языка (RU, EN, ES, CN)
